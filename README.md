## Overview
**Guess the Number** is a beginner-friendly Python console game where the computer randomly selects a number, and the player tries to guess it within a limited number of attempts.  
This project is designed to strengthen basic Python concepts such as loops, conditionals, user input, and random number generation.

---

## Features
- Random number generation
- User input handling
- Feedback on guesses (too high / too low)
- Limited number of attempts
- Simple and interactive console-based gameplay

---

## How the Game Works
1. The program randomly selects a number within a predefined range.
2. The player enters a guess.
3. The program provides feedback:
   - If the guess is too high
   - If the guess is too low
4. The game continues until:
   - The player guesses correctly, or
   - The maximum number of attempts is reached

---

## Technologies Used
- Python 3
- Built-in `random` module
